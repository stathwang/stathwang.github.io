senior thesis

SlopeUnion: An Efficient Algorithm for Product Recommendation in Online Social Rating Networks

SlopeUnion is a new algorithm for efficiently and effectively recommending products in online social ratings network such as Flixster that linearly blends different user and item network structures with the traditional ratings network. The algorithm is easy to implement by an average engineer while maintaining the level of rating prediction accuracy achieved by some of the already existing state-of-the-art recommendation methods.

side projects

microbats: An R Package for Bat Algorithm

Developed an R package for bat algorithm which is a nature-inspired metaheuristic algorithm based on the echolocation behavior of microbats that uses frequency tuning to optimize problems in both continuous and discrete dimensions. The R package makes it easy to implement the standard bat algorithm on any user-supplied function. The algorithm was first developed by Xin-She Yang in 2010.

Clustering Burger King Menu

I used various clustering techniques to find structure in Burger King menu items based on their nutritional values, for I wanted to know which menu items I should certainly avoid and to promote healthy options. Dirichlet process mixture model-based clustering provided the most coherent and robust clusters than any other traditional clustering algorithms such as k-means and hierarchical clustering.

full blog post | Github repo

IMDb Movie Rating Prediction

I looked at the average movie rating and rating variability across different movie genres as well as top 10 popularity ranking of actors, directors, and writers for each genre, and predicted the average rating of a movie using a Bayesian linear mixed model.
